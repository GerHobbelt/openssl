Compiling openssl 1.1.0f for Windows

1.Download openssl 1.1.0f from openssl website
the link: https://www.openssl.org/source/openssl-1.1.0f.tar.gz

2.Run msys for configuration and compiling openssl 

3.Config and compile openssl for Windows
./Configure mingw64 no-shared no-asm zlib --prefix=target-directory
make
make install
